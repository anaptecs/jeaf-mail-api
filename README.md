# JEAF Mail API README #

This repository contains the source code of JEAF Mail API. JEAF Mail provides a so called Mail Service Provider that allows to send mails using the exactly same code in every environment.

## Links ##
For further information please refer to:

* [JEAF Mail](https://anaptecs.atlassian.net/l/c/19P1Kz1H)
* [Maven Repository](https://search.maven.org/artifact/com.anaptecs.jeaf.mail/jeaf-mail-api)
* [Javadoc](https://javadoc.io/doc/com.anaptecs.jeaf.mail/jeaf-mail-api/latest/index.html)

## How do I get set up? ##

* Create a clone of this repository on your local machine.
* Execute Maven on the top level project `maven clean install`
